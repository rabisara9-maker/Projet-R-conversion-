# Convertisseur de Bases en R

Ce projet consiste en une application interactive développée avec **Shiny** en R, permettant de convertir des nombres entre différentes bases (de 2 à 36) ainsi que d'encoder un nombre en "bibi-binaire".

## Fonctionnalités

- Conversion de nombres entre différentes bases (2 à 36).
- Encodage en "bibi-binaire" pour une représentation alternative.
- Interface utilisateur intuitive développée avec Shiny pour une interaction facile.

## Prérequis

Assurez-vous d'avoir installé R et les bibliothèques nécessaires avant d'exécuter ce projet :

- **R** 
- **Shiny** : install.packages("shiny")
